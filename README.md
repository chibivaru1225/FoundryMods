# NoLimitShipping — FOUNDRY Mod

宇宙ステーションへの輸送制限を撤廃し、保管容量を大幅に拡張する FOUNDRY 用 Mod です。

## 機能

- **全アイテム輸送可能** — サイエンスパックや建設セットなど、通常は取引不可のアイテムも含め、隠しアイテム以外の全アイテムをシッピングパッドで宇宙ステーションに送れるようにします。
- **保管容量 ×10000** — 全カテゴリのステーション保管容量を 10000 倍に拡張。大量生産しても輸送船がちゃんと来るようになります。
- **保管庫に全アイテム表示** — 本来カテゴリを持たないアイテム（サイエンスパック等）も、ステーション保管庫の UI スロットに表示されます。

## インストール

1. FOUNDRY Mod Kit を Unity Editor で開く
2. Unity Editor でビルド
3. 出力された Mod を `FOUNDRY/Mods/` フォルダに配置
